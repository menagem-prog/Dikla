איך להפעיל את האפליקציה בטלפון של דקלה
==========================================

יש שתי דרכים. שתיהן דורשות שהאפליקציה תהיה באינטרנט (כתובת URL).

─── שלב 1: להעלות לאינטרנט (בחירה אחת) ───

א. Netlify Drop (הכי פשוט, חינם)
   1. גלוש לאתר: https://app.netlify.com/drop
   2. גרור את כל תיקיית "Dikla Web" (או את כל הקבצים שבתוכה) לתוך החלון.
   3. Netlify יתן לך קישור, למשל: https://שם-אקראי.netlify.app
   4. שמור את הקישור – זה הכתובת של האפליקציה.

ב. GitHub Pages
   1. צור חשבון ב-GitHub (אם אין).
   2. צור "repository" חדש, העלה אליו את כל קבצי תיקיית Dikla Web.
   3. בהגדרות ה-repository: Settings → Pages → Source: Deploy from branch.
   4. בחר את הענף (branch) ושמור. אחרי דקה יופיע קישור כמו:
      https://שם-המשתמש.github.io/שם-הפרויקט

ג. שרת אחר
   אם יש לך דומיין או שרת, העלה את כל תיקיית Dikla Web לשורש האתר כך ש-index.html ייפתח בכתובת הראשית.

─── שלב 2: לפתוח בטלפון ו"להתקין" כמו אפליקציה ───

אחרי שיש לך קישור (מ-Netlify, GitHub או שרת):

• ב-iPhone (Safari):
  1. פתח את הקישור בדפדפן Safari.
  2. הקש על כפתור "שתף" (הריבוע עם החץ למעלה).
  3. גלול ובחר "הוסף למסך הבית" / "Add to Home Screen".
  4. אשר. יופיע אייקון על המסך – לחיצה תפתח את האפליקציה במסך מלא.

• ב-Android (Chrome):
  1. פתח את הקישור בדפדפן Chrome.
  2. בתפריט (שלוש נקודות) בחר "הוסף למסך הבית" / "Add to Home screen" או "התקן אפליקציה".
  3. אשר. יופיע אייקון על המסך – לחיצה תפתח כמו אפליקציה.

─── אייקון לאפליקציה ───

כדי שיהיה אייקון ייעודי (במקום תמונת dikla.jpg):
  הוסף לתיקייה שני קבצים: icon-192.png (192×192 פיקסלים) ו-icon-512.png (512×512).
  אם אין – האפליקציה תשתמש ב-dikla.jpg כאיקון.

─── סיכום ───

1. העלה את תיקיית Dikla Web ל-Netlify Drop (או ל-GitHub Pages).
2. שלח לדקלה את הקישור.
3. היא פותחת את הקישור בטלפון ובוחרת "הוסף למסך הבית".
4. מעכשיו היא פותחת את האייקון מהמסך הראשי – כמו אפליקציה.
